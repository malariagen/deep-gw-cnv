# assets

Static reference files committed to the repository.

| File | Description |
|---|---|
| `PlasmoDB-54_Pfalciparum3D7.gff` | *P. falciparum* 3D7 gene annotations (PlasmoDB release 54) — used to map HMM segments to gene coordinates |
| `20260313_full_cnv_data_pf9.tsv` | Pf9 ground-truth CNV calls (GATK) — used as evaluation reference |
| `Pf_9_samples_20260227.txt` | Pf9 sample manifest with metadata (country, year, study) |
| `core-genome-onebased.bed` | Core genome regions (1-based BED) — excludes hypervariable and repetitive regions |
| `core-genome.interval_list` | Same regions in GATK interval_list format |
| `paths_to_bams_crams.tsv` | BAM/CRAM file paths for the 53,973 Pf9 samples |
| `regions-20130225.onebased.txt` | Legacy region file (1-based) |
| `screenshots/` | App screenshots embedded in the root README |
